# Clicker Box 🖱️📱

Clicker Box is a lightweight, responsive, and sleekly styled incremental counter application built using Python and the Kivy framework. Designed with a modern, high-contrast dark theme, it features dynamic progress animation, real-time percentage updates, and mobile-optimized input validation for setting custom click targets.

---

## ✨ Features

* **Sleek Custom UI:** A beautiful matte-dark interface accented with vibrant neon greens, blues, and gold indicators.
* **Smart Progress Tracking:** Includes an animated progress bar that dynamically updates its color as you approach your target goal.
* **Mobile-Safe Input:** Tailored text input handling that bypasses native mobile clipping issues, ensuring text remains visible and responsive on desktop and Android alike.
* **Instant Feedback:** Features interactive button state animations and screen flashes upon successful increments.
* **Keyboard Hotkeys:** Built-in desktop shortcuts for quick, uninterrupted testing and gameplay.

---

## 🕹️ Controls & Shortcuts

| Action | Desktop Shortcut | UI Interaction |
| :--- | :--- | :--- |
| **Increment Count** | `SPACEBAR` | Click **COUNT** Button |
| **Reset Counter** | `R` or `r` | Click **RESTART** Button |
| **Submit New Goal** | `ENTER` | Click **SET** Button |
